# Blockchain Voting App

This is a simple blockchain voting web application built using Next.js (React) and Solidity.

The application allows users to vote for candidates and view results.

Note: This project is a simulation for coursework purposes. No real blockchain transactions are used.

---

## How to Run

1. Open terminal in the project folder

2. Install dependencies:
npm install

3. Run the project:
npm run dev

4. Open in browser:
http://localhost:3000

---

## Features

- Simulated wallet connection
- Voting system for candidates
- One vote per session
- Live vote results display

---

## Smart Contract

The project includes a Solidity smart contract (Voting.sol) that demonstrates how voting would work on a blockchain.
